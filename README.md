# Claude Code Starter

A starter project for experimenting with [Claude Code](https://claude.ai/code) — Anthropic's official CLI for Claude.

## Getting Started

1. Install Claude Code: `npm install -g @anthropic-ai/claude-code`
2. Run `claude` in this directory to start a session
3. Experiment away

## What is Claude Code?

Claude Code is an agentic coding tool that lives in your terminal. It reads your codebase, edits files, runs commands, and helps you ship faster.
